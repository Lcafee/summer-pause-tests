# L Cafe — Find Your Summer Pause

بازطراحی موبایل‌محور کمپین تابستانی L Cafe با جهت هنری **Cinematic Product-First**.

## اجرای محلی

سایت بدون Build و بک‌اند اجرا می‌شود. برای تست دقیق‌تر، پوشه را با یک وب‌سرور ساده باز کنید:

```bash
python -m http.server 8080
```

سپس آدرس `http://localhost:8080` را باز کنید. بازکردن مستقیم `index.html` نیز پشتیبانی می‌شود.

## انتشار روی GitHub Pages

محتویات همین پوشه را در ریشه مخزن قرار دهید. در تنظیمات GitHub Pages، منبع را روی شاخه `main` و پوشه `/(root)` بگذارید.

## ساختار

- `index.html`: ساختار معنایی رابط
- `assets/css/styles.css`: سیستم بصری و واکنش‌گرایی
- `assets/js/data.js`: اطلاعات محصولات، قیمت‌ها، لاین‌ها و نگاشت کوییز
- `assets/js/app.js`: منطق صفحات، کوییز، نتیجه، منو و دسترس‌پذیری
- `assets/images/`: تصاویر اصلی ۸ محصول
- `assets/brand/lcafe-logo.svg`: لوگوی رسمی L Cafe
- `QA_REPORT.md`: گزارش تست و کنترل کیفیت

## ویرایش محتوا

اطلاعات محصولات و قیمت‌ها فقط در `assets/js/data.js` نگهداری می‌شوند. مسیر عکس هر محصول نیز همان‌جا تعریف شده است.

## منطق کوییز

کوییز دو سؤال دارد. سؤال اول لاین نوشیدنی و سؤال دوم محصول اصلی را تعیین می‌کند. نتیجه اصلی برای هر مسیر ثابت است.

**پیشنهاد ثانویه عمداً تصادفی است**، در هر اجرای نتیجه می‌تواند تغییر کند و همیشه با محصول اصلی متفاوت می‌ماند.

## فونت

برای جلوگیری از وابستگی خارجی، حفظ سرعت و اطمینان از مجوز استفاده، پروژه از فونت‌های سیستمی با Fallback مشخص استفاده می‌کند و هیچ سرویس فونت خارجی بارگذاری نمی‌شود.
