# L Cafe — Find Your Summer Pause

نسخه نهایی و بهینه‌شده برای انتشار مستقیم روی `GitHub Pages`.

این نسخه همان طراحی `Cinematic Product-First` تأییدشده را حفظ می‌کند. تصاویر با نسخه‌های اصلی باکیفیت جایگزین شده‌اند و ساختار، رنگ‌ها، چیدمان، متن‌ها و انیمیشن‌ها بدون تغییر باقی مانده‌اند. تنها سیستم فونت برای نمایش یکدست‌تر در موبایل به فونت‌های وب استاندارد تغییر کرده است.

## انتشار روی GitHub Pages

محتویات همین پوشه را مستقیماً در ریشه مخزن قرار دهید؛ یعنی فایل `index.html` باید در صفحه اول مخزن دیده شود.

سپس در GitHub:

1. وارد `Settings` شوید.
2. بخش `Pages` را باز کنید.
3. گزینه `Deploy from a branch` را انتخاب کنید.
4. شاخه `main` و پوشه `/(root)` را انتخاب کنید.
5. روی `Save` بزنید.

فایل `.nojekyll` نیز داخل پروژه قرار گرفته تا پروژه بدون پردازش اضافی Jekyll منتشر شود.

## اجرای محلی

سایت بدون Build، بک‌اند یا نصب پکیج اجرا می‌شود:

```bash
python -m http.server 8080
```

سپس این آدرس را باز کنید:

```text
http://localhost:8080
```

## ساختار پروژه

- `index.html` — ساختار اصلی رابط
- `assets/css/styles.css` — تمام استایل‌ها و واکنش‌گرایی
- `assets/js/data.js` — محصولات، قیمت‌ها، تصاویر و نگاشت کوییز
- `assets/js/app.js` — منطق کوییز، نتیجه، منو و دسترس‌پذیری
- `assets/images/` — تصاویر بهینه‌شده ۸ محصول
- `assets/brand/lcafe-logo.svg` — لوگوی رسمی
- `QA_REPORT.md` — گزارش کنترل کیفیت نسخه نهایی
- `.nojekyll` — آماده‌سازی انتشار مستقیم روی GitHub Pages

## تصاویر محصولات

هر ۸ تصویر از فایل‌های اصلی `1254×1254px` جایگزین شده‌اند.

برای حفظ کیفیت و کاهش حجم، تصاویر به `WebP` با کیفیت بالا تبدیل شده‌اند:

- Crop تصاویر تغییر نکرده است.
- نسبت لیوان‌ها تغییر نکرده است.
- رنگ و نور عکس‌ها دست‌کاری نشده است.
- ابعاد واقعی تصویر حفظ شده است.
- تصاویر منوی کامل به‌صورت `Lazy Load` بارگذاری می‌شوند.
- تصاویر ضروری صفحه ورود از قبل `Preload` می‌شوند.
- ابعاد تصاویر در HTML مشخص شده‌اند تا `Layout Shift` رخ ندهد.

فایل‌های PNG اولیه در خروجی انتشار قرار نگرفته‌اند تا حجم مخزن و زمان دانلود کاربران افزایش پیدا نکند.

## ویرایش محتوا

اطلاعات محصولات، قیمت‌ها و مسیر تصاویر در فایل زیر قرار دارند:

```text
assets/js/data.js
```

نام فایل تصاویر به‌صورت حروف کوچک ثبت شده است تا در GitHub Pages و سیستم‌های حساس به بزرگی و کوچکی حروف مشکلی ایجاد نشود.

## منطق کوییز

کوییز دو سؤال دارد:

- سؤال اول لاین نوشیدنی را مشخص می‌کند.
- سؤال دوم محصول اصلی را تعیین می‌کند.
- نتیجه اصلی هر مسیر ثابت است.
- پیشنهاد ثانویه عمداً تصادفی است و همیشه با نتیجه اصلی تفاوت دارد.

## وابستگی‌ها

پروژه:

- تصاویر، لوگو، CSS و JavaScript کاملاً محلی هستند.
- فقط فونت‌های متن‌باز از Google Fonts بارگذاری می‌شوند.
- در صورت قطع دسترسی به سرویس فونت، Fallbackهای سیستمی فعال می‌شوند.
- API یا بک‌اند ندارد.
- Build Step ندارد.
- مستقیماً روی GitHub Pages اجرا می‌شود.

## فونت‌های موبایل

برای نمایش یکدست‌تر روی iOS و Android، سایت از سه فونت متن‌باز Google Fonts استفاده می‌کند:

- `Vazirmatn` برای متن فارسی
- `Roboto Condensed` برای برچسب‌ها و متن لاتین باریک
- `Libre Baskerville` برای تیترهای لاتین سینمایی

فونت‌ها با `display=swap` بارگذاری می‌شوند و در صورت قطع دسترسی، فونت‌های سیستمی جایگزین خواهند شد.
