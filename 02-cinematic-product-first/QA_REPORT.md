# گزارش کنترل کیفیت نسخه HQ و بهینه‌شده

## محدوده تغییرات

در این مرحله فقط موارد زیر انجام شدند:

- جایگزینی تمام ۸ عکس کم‌حجم قبلی با تصاویر اصلی و باکیفیت ارسالی
- تبدیل تصاویر اصلی به نسخه وب بهینه
- اصلاح مسیر فایل‌های تصاویر از `JPG` به `WebP`
- تعیین ابعاد واقعی تصاویر
- افزودن `decoding="async"` برای تصاویر Raster
- فعال‌سازی `Lazy Loading` برای تصاویر منوی کامل
- افزودن نوع فایل به Preload تصاویر ضروری
- افزودن فایل `.nojekyll` برای انتشار مستقیم روی GitHub Pages

هیچ تغییری در جهت هنری، ساختار صفحات، رنگ‌ها، تصاویر، فاصله‌ها، متن‌ها یا Motion سایت ایجاد نشده است. تنها متغیرهای خانواده فونت و لینک بارگذاری فونت‌های وب تغییر کرده‌اند.

## تصاویر جدید

تمام تصاویر ورودی:

- تعداد: ۸ فایل
- ابعاد: `1254×1254px`
- نسبت: `1:1`
- فضای رنگی: `RGB`
- فرمت اولیه: `PNG`

خروجی وب:

- تعداد: ۸ فایل
- ابعاد: `1254×1254px`
- فرمت: `WebP`
- کیفیت Encoding: `92`
- حجم کل PNGهای اولیه: `15,723,037 bytes`
- حجم کل تصاویر WebP نهایی: `1,005,482 bytes`
- کاهش حجم: حدود `93.61%`

در فرایند تبدیل هیچ Crop، Resize، کشیدگی، تغییر نسبت یا اصلاح رنگی روی تصاویر انجام نشده است.

## کنترل عدم تغییر ظاهر

برای اطمینان از ثابت‌ماندن ساختار بصری:

- فایل تصاویر، داده‌ها و منطق JavaScript با نسخه HQ قبلی بایت‌به‌بایت یکسان باقی ماندند.
- تنها سه متغیر خانواده فونت در CSS تغییر کردند.
- `SHA-256` فایل CSS نهایی:

```text
467dd5a9a2030cfd834c1d258bd7e67115f1fcd39716fd2c560d911e58272998
```

مختصات و ابعاد عناصر اصلی صفحه شروع و کوییز در عرض `390px` با نسخه قبلی مقایسه شدند و اختلاف Layout برابر صفر بود.

## اندازه‌های تست‌شده

نسخه نهایی روی اندازه‌های زیر بررسی شد:

- `320 × 620`
- `320 × 700`
- `360 × 740`
- `375 × 667`
- `390 × 844`
- `412 × 915`
- `430 × 932`
- Landscape: `844 × 390`

در تمام اندازه‌ها:

- صفحه شروع سالم بود.
- کوییز سالم بود.
- منوی کامل سالم بود.
- سرریز افقی مشاهده نشد.
- کنترل‌های اصلی خارج از صفحه قرار نگرفتند.

## مسیرهای کوییز تست‌شده

تمام ۸ نتیجه اصلی اجرا و بررسی شدند:

- `IT-01` — آیس‌تی اپل پارادایس
- `IT-02` — آیس‌تی پیچ بلک
- `RF-02` — رفرشر انبه و شاهتوت
- `RF-03` — رفرشر جینجر بیزل
- `CL-01` — کلودی پسته
- `CL-03` — کلودی کوکی
- `FR-01` — فراپه کارامل نمکی
- `FR-02` — فراپه ارده

برای هر ۸ مسیر:

- کد نتیجه صحیح بود.
- تصویر صحیح نمایش داده شد.
- فرمت تصویر `WebP` بود.
- ابعاد واقعی تصویر `1254×1254px` بود.
- قیمت صحیح نمایش داده شد.
- پیشنهاد ثانویه با پیشنهاد اصلی متفاوت بود.
- خطای اجرایی مشاهده نشد.

## پیشنهاد ثانویه تصادفی

یک نتیجه ثابت در ۱۰ اجرای جداگانه بررسی شد:

- پیشنهاد اصلی هیچ‌گاه تکرار نشد.
- چند پیشنهاد ثانویه متفاوت تولید شد.
- رفتار تصادفی قبلی حفظ شده است.

## منوی کامل

- هر ۴ Chapter ساخته و بررسی شد.
- هر ۴ دکمه ناوبری Chapter کار کردند.
- هر ۸ محصول نمایش داده شدند.
- هر ۸ تصویر منو با ابعاد `1254×1254px` بارگذاری شدند.
- تمام تصاویر منو دارای `loading="lazy"` هستند.
- نام، کد، قیمت، توضیح و تگ‌ها تغییری نکرده‌اند.

## حالت‌های خطا

- نبود تصویر محصول: تصویر جایگزین داخلی به‌درستی نمایش داده شد.
- نبود قیمت: قیمت جایگزین `۲۲۰٬۰۰۰ تومان` به‌درستی نمایش داده شد.
- مسیر تمام فایل‌های محلی بررسی شد.
- هیچ فایل مفقودی در خروجی نهایی وجود ندارد.

## اجرای استاتیک و GitHub Pages

- ریشه سایت از وب‌سرور استاتیک پاسخ `HTTP 200` دریافت کرد.
- فایل CSS پاسخ `HTTP 200` دریافت کرد.
- تصویر اصلی صفحه ورود پاسخ `HTTP 200` دریافت کرد.
- تمام مسیرها Relative هستند.
- هیچ API یا بک‌اندی وجود ندارد.
- تصاویر، لوگو، CSS و JavaScript محلی‌اند.
- تنها درخواست خارجی مربوط به فونت‌های متن‌باز Google Fonts است.
- فایل `.nojekyll` اضافه شده است.
- ساختار خروجی برای قرارگیری در ریشه مخزن آماده است.

مرورگر Sandbox اجازه ناوبری مستقیم به `localhost` و `file://` را نمی‌داد؛ تست کامل مرورگر با همان فایل‌های واقعی و یک Origin محلی شبیه‌سازی‌شده انجام شد و پاسخ HTTP فایل‌ها نیز جداگانه با وب‌سرور استاتیک کنترل شد.

## نتیجه نهایی

- ساختار و جهت هنری سایت تغییر نکرده است؛ تایپوگرافی موبایل یکدست‌تر شده است.
- کیفیت عکس‌ها به‌صورت محسوس بالاتر است.
- حجم ۸ تصویر نسبت به PNGهای اصلی بیش از ۹۳ درصد کاهش یافته است.
- بارگذاری منوی کامل سبک‌تر شده است.
- تمام ۸ مسیر نتیجه سالم‌اند.
- خطای Console یا JavaScript مشاهده نشد.
- پروژه آماده آپلود مستقیم روی GitHub Pages است.

## کنترل فونت موبایل

- فونت فارسی به `Vazirmatn` تغییر کرد.
- تیترهای لاتین به `Libre Baskerville` تغییر کردند.
- برچسب‌های لاتین به `Roboto Condensed` تغییر کردند.
- `preconnect` برای دامنه‌های فونت اضافه شد.
- `font-display: swap` از طریق Google Fonts فعال است.
- Fallbackهای سیستمی برای حالت آفلاین حفظ شدند.

- Landscape با ابعاد `844×390` بررسی شد.
- تست با فونت‌های محلی مشابه از نظر متریک انجام شد.

- در صورت بارگذاری‌نشدن فونت خارجی، Fallbackهای سیستم باعث توقف یا خرابی سایت نمی‌شوند.
