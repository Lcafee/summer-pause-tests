# L Cafe — Find Your Summer Pause

سایت استاتیک و موبایل‌محور کمپین تابستانی `L Cafe` با جهت هنری **Contemporary Art Collage**.

## نسخه مخصوص GitHub Pages

برای انتشار سریع، از فایل زیر استفاده کنید:

```text
L-Cafe-Summer-Pause-GitHub-Pages.zip
```

این بسته فقط فایل‌های لازم Runtime را دارد و `index.html` مستقیماً در ریشه آن قرار گرفته است. تصاویر اصلی سنگین و فایل‌های تست در بسته انتشار قرار نگرفته‌اند.

## انتشار روی GitHub Pages

1. فایل ZIP مخصوص GitHub Pages را Extract کنید.
2. **محتویات داخل پوشه** را در ریشه Repository آپلود کنید؛ خود پوشه مادر یا فایل ZIP را آپلود نکنید.
3. بعد از Upload باید ساختار ریشه Repository این‌طور باشد:

```text
index.html
.nojekyll
README.md
assets/
```

4. تغییرات را `Commit` و `Push` کنید.
5. در GitHub وارد این مسیر شوید:

```text
Settings > Pages
```

6. در بخش `Build and deployment` انتخاب کنید:

```text
Source: Deploy from a branch
Branch: main
Folder: / (root)
```

7. روی `Save` بزنید و چند دقیقه صبر کنید.

تمام مسیرها Relative هستند؛ بنابراین سایت هم در دامنه اصلی و هم در آدرس Project Pages مانند مسیر زیر درست اجرا می‌شود:

```text
https://username.github.io/repository-name/
```

## نکات مهم Upload

- نام پوشه‌ها و فایل‌ها را تغییر ندهید؛ GitHub Pages به کوچک و بزرگ بودن حروف حساس است.
- فایل `.nojekyll` باید باقی بماند.
- `index.html` نباید داخل یک پوشه اضافی قرار بگیرد.
- برای جایگزینی نسخه قبلی، همه فایل‌های قبلی سایت را حذف و محتویات این بسته را جایگزین کنید.
- پس از انتشار، روی گوشی یک‌بار صفحه را با Hard Refresh یا در حالت Incognito باز کنید تا Cache نسخه قبلی نمایش داده نشود.

## بهینه‌سازی موبایل

نسخه فعلی برای عرض‌های `320px` تا `430px` و ارتفاع‌های کوتاه و بلند تنظیم شده است:

- هر دو مسیر اصلی در اولین Viewport گوشی دیده می‌شوند.
- Safe Area بالا و پایین آیفون رعایت شده است.
- از `svh` و `dvh` برای نوار متغیر مرورگر موبایل استفاده شده است.
- در کوییز، محتوای نمایشگرهای بسیار کوتاه قابل اسکرول باقی می‌ماند.
- هیچ سرریز افقی در عرض‌های تست‌شده وجود ندارد.
- رابط در مسیر زیرمجموعه GitHub Pages نیز تست شده است.

## تغییر اطلاعات و قیمت‌ها

تمام اطلاعات اجرایی محصولات، قیمت‌ها، سؤال‌ها و نگاشت نتایج در فایل زیر قرار دارد:

```text
assets/js/data.js
```

پیشنهاد اصلی هر مسیر ثابت است؛ پیشنهاد ثانویه عمداً تصادفی است و با محصول اصلی یکسان نمی‌شود.

## اجرای محلی

فایل `index.html` را مستقیماً باز کنید یا از یک سرور ساده استفاده کنید:

```bash
python -m http.server 8000
```

سپس:

```text
http://localhost:8000
```

## فونت‌های محلی

فونت‌ها داخل `assets/fonts/` قرار دارند و سایت به سرویس خارجی وابسته نیست:

- `Vazirmatn` برای متن فارسی و رابط
- `Cormorant Garamond` برای تیترهای نمایشی لاتین
- `Barlow Condensed` برای جزئیات ادیتوریال و شماره‌ها

هر سه خانواده تحت مجوز `SIL Open Font License 1.1` استفاده شده‌اند. متن مجوزها در `assets/fonts/licenses/` قرار دارد.
