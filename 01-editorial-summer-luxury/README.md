# L Cafe — Find Your Summer Pause

نسخه نهایی آماده انتشار مستقیم روی GitHub Pages.

## انتشار روی GitHub Pages

1. یک Repository جدید در GitHub بسازید.
2. **تمام محتویات همین پوشه** را در ریشه Repository آپلود کنید؛ فایل `index.html` باید در ریشه باشد.
3. از مسیر `Settings → Pages`، گزینه `Deploy from a branch` را انتخاب کنید.
4. Branch را روی `main` و Folder را روی `/ (root)` قرار دهید و Save کنید.

هیچ Build، Backend، npm یا CDN لازم نیست.

## اجرای محلی

ساده‌ترین روش، بازکردن مستقیم `index.html` است. برای تست مشابه GitHub Pages:

```bash
python -m http.server 8080
```

سپس `http://localhost:8080` را باز کنید.

## ساختار پروژه

```text
index.html
assets/
  css/styles.css
  js/data.js
  js/app.js
  fonts/
  images/
    products/         تصاویر بهینه‌شده و باکیفیت مورد استفاده سایت
    source-products/  فایل‌های PNG اصلی و دست‌نخورده
content/
.nojekyll
```

## تغییر محصولات و قیمت‌ها

تمام نام‌ها، توضیحات، تگ‌ها، مسیر عکس‌ها، قیمت‌ها و منطق دو سؤال در فایل زیر هستند:

```text
assets/js/data.js
```

قیمت به تومان و به‌صورت عدد صحیح وارد می‌شود.

## تغییر عکس‌ها

نسخه‌ای که سایت بارگذاری می‌کند در این مسیر است:

```text
assets/images/products/
```

نام فایل‌ها باید با مسیر ثبت‌شده در `assets/js/data.js` یکسان بماند. فایل‌های اصلی با کیفیت کامل در `assets/images/source-products/` نگهداری شده‌اند.

## فونت‌ها

فونت‌ها کاملاً محلی و با فرمت `WOFF2` هستند و از سرویس خارجی دریافت نمی‌شوند:

- Vazirmatn برای متن و رابط فارسی
- Estedad برای تیترهای فارسی
- Cormorant Garamond برای تایپوگرافی لاتین ادیتوریال

تعریف فونت‌ها در ابتدای `assets/css/styles.css` و فایل‌های آن‌ها در `assets/fonts/` قرار دارند. مجوزهای OFL نیز کنار فونت‌ها موجودند.

## منطق کوییز

کوییز دقیقاً دو سؤال دارد. پاسخ سؤال اول لاین را انتخاب می‌کند و پاسخ سؤال دوم محصول اصلی را تعیین می‌کند. پیشنهاد ثانویه **عمداً تصادفی** است، ولی هیچ‌گاه با پیشنهاد اصلی یکسان نمی‌شود.
