# گزارش کنترل کیفیت و بهینه‌سازی

## نتیجه نهایی

- تمام `32` مسیر کوییز تست شد: ۸ نتیجه در هر یک از عرض‌های `320`، `360`، `390` و `430px`
- بدون خطای JavaScript، Console یا فایل محلی مفقود
- بدون سرریز افقی در صفحه اول، کوییز، نتیجه و منوی کامل
- عملکرد صحیح با لمس و کیبورد
- عملکرد صحیح بازگشت به سؤال قبل و شروع مجدد
- نمایش هر ۸ محصول، قیمت و تصویر صحیح
- پیشنهاد ثانویه در تمام تست‌ها با پیشنهاد اصلی متفاوت بود
- حالت نبود قیمت و تصویر جایگزین تست شد
- `prefers-reduced-motion` تست شد
- بارگذاری مستقیم و ساختار مناسب GitHub Pages حفظ شد

## قفل ظاهری

نسخه قبل و بعد در عرض مرجع `390px` و با شرایط فونت یکسان، از نظر مختصات و ابعاد عناصر اصلی مقایسه شدند:

- صفحه اول
- سؤال اول و دوم
- صفحه نتیجه
- منوی کامل

بیشترین اختلاف هندسی ثبت‌شده: **`0px`**

هیچ تغییر عمدی در رنگ، تایپوگرافی، فاصله‌ها، چیدمان، سایه‌ها، حرکت‌ها یا محتوای رابط ایجاد نشده است.

## کاهش بار اولیه

اندازه منابع محلی دریافت‌شده در بار اولیه‌ی عرض `390px` با تراکم پیکسلی `3x`، با حذف پاسخ خارجی Estedad از هر دو اندازه‌گیری:

| معیار | قبل | بعد | کاهش |
|---|---:|---:|---:|
| بار اولیه منابع محلی | 738,483 B | 275,047 B | 62.76% |
| سه تصویر صفحه اول | 644,798 B | 161,834 B | 74.90% |
| CSS اجرایی | 33,820 B | 20,694 B | 38.81% |
| JavaScript اجرایی | 35,839 B | 28,941 B | 19.25% |
| لوگوی SVG | 23,242 B | 17,242 B | 25.81% |

در عرض‌های `320` تا `390px` بار اولیه حدود `275 KB` است. در عرض `430px` به‌دلیل انتخاب تصاویر بزرگ‌تر و متناسب با تراکم `3x` حدود `323 KB` است.

## بهینه‌سازی تصاویر

- حفظ هشت فایل اصلی `1254×1254`
- ساخت نسخه‌های `480`، `640`، `960` و `1100px`
- ساخت نسخه‌ی ویژه `850px` برای تصویر اصلی صفحه اول
- استفاده از `srcset` و `sizes` برای انتخاب خودکار فایل
- `fetchpriority="high"` فقط برای تصویر قهرمان
- Lazy Loading برای تصاویر منو و پیشنهاد ثانویه
- شروع Decode تصویر نتیجه در مرحله مکث، پیش از نمایش نتیجه

حجم مجموع هشت تصویر منو در موبایل مرجع `390px / 3x`:

- قبل: `1,723,054 B`
- بعد: `747,388 B`
- کاهش: `56.62%`

## بهینه‌سازی کد و منابع

- حذف CSS مربوط به اجزای حذف‌شده و سلکتورهای مرده
- حذف توابع و Observerهای دسته‌بندی منسوخ
- تبدیل جست‌وجوی محصول از پیمایش آرایه به `Map`
- Cache کردن `Intl.NumberFormat`
- ترکیب `catalog.js` و `app.js` در یک فایل Production
- بارگذاری JavaScript با `defer`
- Minify کردن CSS و JavaScript
- محلی‌کردن و Subset کردن Cormorant Garamond
- بهینه‌سازی SVG لوگو
- حذف فایل PNG لوگوی استفاده‌نشده
- حذف فایل‌های Minify میانی و تکراری
- حذف درخواست اولیه‌ی تصویر Placeholder با Data URI یک‌پیکسلی
- حفظ سورس خوانا در کنار فایل‌های Production
