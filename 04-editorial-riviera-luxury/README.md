# L Cafe — Summer Pause

نسخه نهایی آماده انتشار روی GitHub Pages.

## انتشار

1. فایل ZIP را Extract کن.
2. تمام فایل‌ها و پوشه‌های داخل آن را مستقیماً در ریشه Repository قرار بده؛ `index.html` باید در ریشه باشد.
3. در GitHub وارد `Settings → Pages` شو.
4. بخش `Build and deployment` را روی `Deploy from a branch` قرار بده.
5. Branch را روی `main` و Folder را روی `/ (root)` بگذار و `Save` کن.

## فونت‌ها

هر دو خانواده فونت به‌صورت محلی همراه سایت قرار دارند:

- `Estedad` برای فارسی و رابط کاربری
- `Cormorant Garamond` برای تیترها و جزئیات انگلیسی

سایت برای نمایش فونت به Google Fonts یا سرویس خارجی وابسته نیست.

## فایل‌های اجرایی

- `index.html`
- `styles/main.min.css`
- `scripts/site.min.js`
- `assets/`

فایل `.nojekyll` نیز برای انتشار مستقیم و بدون پردازش Jekyll قرار داده شده است.
